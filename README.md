# DroneOptimization
This project is a MATLAB-based physics and engineering simulation tool for analyzing a quadcopter drone design. It evaluates mass properties, thrust performance, aerodynamic stability, PID control response, and structural resonance to help iterate safe and efficient drone configurations.

How to Run
1. Open MATLAB
2. Save the script as: "drone_simulation.m"
3. Click Run or execute in command window: "drone_simulation"

 
**The script generates:

Console Output:
  Total drone mass,
  Center of gravity,
  Inertia tensor values,
  Thrust-to-weight evaluation,
  Resonance safety warnings,
  
Plots:
  Pitch Stabilization vs Time,
  Shows PID response under wind disturbance,
  Frequency Response Curve,
  Shows resonance amplification vs motor RPM,
  Highlights dangerous operating zones,
